HTML Table Elements Reference

Element Tag Description
Table <table> Defines the entire table structure.
Table Row <tr> Defines a row within the table.
Table Header Cell <th> Defines a header cell, usually bold and centered by default.
Table Data Cell <td> Defines a standard data cell.
Table Head <thead> Groups header content in a table.
Table Body <tbody> Groups the main content of the table.
Table Footer <tfoot> Groups footer content in a table.
Column Group <colgroup> Defines a group of one or more columns for formatting.
Column <col> Specifies column properties within a colgroup.
Caption <caption> Provides a title or explanation for the table.
